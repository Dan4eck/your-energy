## 1. Общее описание

| Параметр | Значение |
|---|---|
| Название | Sculpt your body & mind |
| Тип сайта | Landing Page (одностраничник) с якорным меню |
| Цель | Привлечь участниц в закрытый клуб, показать формат тренировок и экспертов, дать возможность оплатить подписку или записаться |
| Аудитория | Женщины, интересующиеся комплексным подходом к телу и психике |
| Тональность | Спокойная, уверенная, заботливая — без агрессивных sales-триггеров |

Референс по стилю: https://take-your-time.ru

## 4. Анализ стиля (по референсу)

### Цвета
- Фон: тёплый пастельно-бежевый / кремовый
- Акценты: оливковый / приглушённый зелёный для кнопок CTA
- Текст: чёрный или очень тёмно-коричневый
- Вторичный: серый для вспомогательных надписей

### Типографика
- Заголовки: крупный serif (Cormorant) — элегантность ✅ реализовано
- Текст: чистый sans-serif (Inter) — читаемость ✅ реализовано
- Декор: курсив для акцентных фраз ✅ реализовано

### Фото
- Hero: вертикальное, художественное, телесное ✅ hero-yoga.jpg
- Автор/эксперты: живые, тёплые, естественные цвета ✅ meditation.jpg

### UI-элементы
- Кнопки: небольшие, с мягким скруглением, приглушённые цвета ✅ реализовано
- Карточки: без тяжёлых теней, либо с мягкой тенью ⏳ заготовки есть
- Аккордеон: для FAQ — минималистичный, с тонкими разделителями ⏳ ожидает
- Слайдер: для отзывов (если много) ⏳ ожидает

### Анимации
- Плавный scroll-to-anchor ✅ реализовано
- Лёгкие fade-in / parallax при скролле секций ⏳ ожидает
- Микроанимация кнопок (hover — смена цвета или подчёркивание) ✅ реализовано

## 7. Архитектурные решения

### Масштаб
- MVP: одностраничник с якорями ✅
- Рост: позже можно добавить `/blog`, `/specialists/:slug`, `/workouts`

### Контент
- Тексты и метаданные — в компонентах / JSON / MDX / CMS, чтобы заказчик мог редактировать
- Фото — в `public/` или внешнем CDN (Cloudinary)
- Отзывы и специалисты — лучше через CMS, чтобы каждое изменение не требовало деплоя

### Производительность
- Цель: Lighthouse 90+ по всем параметрам
- Lazy load для изображений и iframe с видео
- Preload шрифтов и hero-изображения
